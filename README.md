# New Wibot

Whatsapp Bot for Weaboo and Anime Lovers.

## Installation

Use the Node Package Manager [npm](https://www.npmjs.com/get-npm) to install New Wibot.

```bash
>git clone https://github.com/ikkehman/new-wibot-anime-whatsapp-bot.git
>npm install
```

## Usage

```npm
npm start
```
